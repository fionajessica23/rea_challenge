# REA Code Challenge


### Commands

```sh
$ yarn install      # install dependencies

$ yarn start        # start app

$ yarn test         # run all test suites
$ yarn jest         # run snapshot tests
$ yarn mocha        # run unit tests

$ yarn build        # build app to the /build directory
```

---

### Technologies used

- React, Redux
- Webpack, Babel
- CSS modules, SASS
- Mocha, Chai, Jest
- ESLint
